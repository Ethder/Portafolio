Tienda Mars

Tienda Mars ha sido desarrollada utilizando un enfoque basado en el patrón de arquitectura MVC (Modelo-Vista-Controlador), lo que permite una separación clara entre la lógica de negocio, la interfaz de usuario y el acceso a los datos. La interfaz fue construida con HTML y estilizada con CSS para ofrecer una experiencia visual atractiva y adaptada a dispositivos móviles. En el lado del servidor, se empleó C# junto con ASP.NET para gestionar la lógica de la aplicación, mientras que los datos de productos, clientes y pedidos se almacenan y administran eficientemente en una base de datos SQL. Gracias a esta combinación de tecnologías, la tienda "Mars" ofrece un rendimiento sólido, escalabilidad y facilidad de mantenimiento.

Pastillas

Mi primer proyecto en C# fue una aplicación sencilla pero significativa, inspirada en la icónica escena de Matrix, donde el usuario debía elegir entre una pastilla roja o una azul, cada una representando un camino diferente dentro del programa. Este ejercicio me permitió familiarizarme con los conceptos básicos de la lógica condicional y la programación orientada a objetos. Años después, decidí mejorar y modernizar ese proyecto utilizando tecnologías web como HTML, CSS y JavaScript. Rediseñé la interfaz para que fuera visualmente más atractiva y responsiva, incorporando animaciones y efectos visuales que reflejan mejor la atmósfera futurista de la película. Esta evolución no solo mejoró la experiencia del usuario, sino que también marcó un avance importante en mi crecimiento como desarrollador.

Ethder en navidad

Ethder es un pequeño juego navideño para Android que desarrollé como mi primer acercamiento al diseño de videojuegos. La historia sigue a Ethder, un nomo decidido que debe rescatar a una galleta mágica perdida en un mundo invernal lleno de sorpresas. El juego consta de un solo nivel, pero está lleno de acción: el objetivo es recolectar la mayor cantidad posible de melcochas mientras se esquivan obstáculos y se avanza por un paisaje navideño encantador. Con gráficos simples pero coloridos y una banda sonora alegre, Ethder ofrece una experiencia breve pero entretenida, ideal para compartir el espíritu navideño de una forma divertida y original.

Android aplicaciones

He desarrollado una serie de aplicaciones móviles utilizando Android y Kotlin, centradas en la gestión y control de medidas estadísticas y operativas a nivel departamental. Estas aplicaciones permiten el registro, análisis y visualización de datos en tiempo real, facilitando la toma de decisiones estratégicas dentro de las organizaciones. Entre sus objetivos principales se encuentran la optimización de procesos, el seguimiento del cumplimiento de metas y la mejora continua mediante indicadores clave de desempeño (KPIs). Las herramientas incluyen funciones de generación de reportes, alertas automatizadas y paneles personalizados, alineándose con los objetivos de eficiencia, transparencia y control en cada área funcional.
